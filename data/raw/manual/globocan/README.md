# GLOBOCAN / Global Cancer Observatory — arquivo manual

1. Acesse: https://gco.iarc.who.int/today/
2. Va em "Data visualization" / exportacao por cancer e pais.
3. Selecione "Oesophagus" como localizacao do cancer.
4. Exporte os dados (CSV ou Excel) de incidencia e mortalidade por pais.
5. Salve o arquivo NESTA PASTA, sem renomear.

Serve como base de TRIANGULACAO/validacao cruzada com o GBD — nao e a
fonte primaria do painel.
