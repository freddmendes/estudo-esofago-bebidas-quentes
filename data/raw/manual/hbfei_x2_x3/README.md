# Componentes X2 e X3 do HBFEI — curadoria manual obrigatoria

Estes dois componentes NAO existem em nenhum banco de dados pronto.
O proprio protocolo (Secao 10.1) exige revisao de literatura para
construi-los. Isso NAO pode ser 100% automatizado — e uma decisao
cientifica, nao uma extracao mecanica.

Preencha o arquivo "template_x2_x3.csv" desta pasta, um pais por linha,
usando os estudos-ancora citados no protocolo como ponto de partida
(Golestan Cohort - Ira; ESCCAPE - Malawi/Tanzania/Quenia; estudos de
caso-controle na China) e expandindo via busca em PubMed/Google
Scholar por outros paises de interesse.

Se quiser, posso te ajudar a rodar buscas e organizar candidatos — mas
a decisao final de qual numero entra na planilha e sempre sua (com o
orientador), com a fonte anotada.
